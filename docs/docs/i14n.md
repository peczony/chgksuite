# Перевод и кастомизация

В chgksuite есть поддержка возможности мультиязычности. Сейчас поддержан только украинский язык. Встроенные файлы для его поддержки находятся в папке `resources`. Узнать точный путь к папке можно, запустив команду:

```bash
python -c "import chgksuite.common; print(chgksuite.common.get_source_dirs()[1])"
```

## Парсинг пакета на другом языке

![](images/i14n_parse.png)

Нажмите на кнопку «Открыть файл» рядом с опцией «Файл с регулярными выражениями» и выберите `regexes_ua.json`. Откройте файл с пакетом по-украински.

## Экспорт пакета на другом языке

Нажмите на кнопку «Открыть файл» рядом с опцией «Конфиг для интернационализации» и выберите `regexes_ua.json`. Откройте 4s-файл. В получившемся файле названия полей «Вопрос», «Ответ» и т.д. будут по-украински.

![](images/i14n.png)

## Кастомизация полей

Если в одном конкретном вопросе одно конкретое поле хочется переименовать (например, в одном из вопросов «Вопрос» в «Загадка» или «Автор» в «Авторка»), можно написать в 4s что-то типа `? !!Загадка Один журналист отмечает...` или `@ !!Авторка Мария Подрядчикова`, и эти названия сохранятся в выходном файле.
