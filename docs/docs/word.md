# Word

![](images/word.png)

Откройте **4s-файл**, который у вас получился после [парсинга](/4s). По умолчанию при экспорте ответы будут забелены — это полезно, если вы отправляете файл на текстовое тестирование. Если вам это не нужно, например, при сдаче пакета — включите галочку «Не забелять ответы». Остальные галочки говорят сами за себя.

После нажатия кнопки «Запустить» в той же папке, что и 4s-файл, появится файл в формате docx.
